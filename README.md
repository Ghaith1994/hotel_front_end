# hotel_front_end

## Tech-stack
```
- VueJS 2
- TypeScript
- webpack
- yarn or npm
- BEM
- SCSS with flex
```

## Project setup
```
npm install
```

```
edit .env file with backend server api
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
